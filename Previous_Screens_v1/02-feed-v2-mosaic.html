·핵심 규칙: 모든 셀 3:4 비율 절대 고정 (가로:세로). 인앱 카메라 촬영도 동일 비율. 모자이크는 균등 그리드 위주 → n=1: 1x1 / n=2: 2x1 / n=3,4: 2x2 (3은 1빈) / n=5,6: 3x2 (5는 1빈) / n=7,8,9: 3x3 (7은 2빈, 8은 1빈).
+두 가지 빈 칸 구분: (a) 멤버 NO SIGNAL = dashed + 큰 이니셜 + "NO SIGNAL" 라벨 (그 사람이 이번 라운드 무송출), (b) 그리드 슬롯 = 매우 흐릿한 dashed (인원수가 그리드 칸을 다 못 채워 비어있는 자리, 멤버 자체 없음). 시각적으로 두 의미 분리.
+가로 스와이프: 한 라운드 = 2H, 24H = 12 라운드. NOW 가 우측 끝(시간선 → ←), 좌로 스와이프하면 과거. CSS scroll-snap 으로 한 페이지씩 정확히 정렬. 스크롤바 숨김.
+페이지 인디케이터: 12 dot 가로 timeline. 현재 보고 있는 라운드 = 잉크 검정 길쭉한 bar (14px, 2px radius), NOW 라운드(우측 끝) = 그린 LED 호흡 펄스. 사라질 라운드 (-22H) = 비어있는 ring. 그 아래 ROUND 시간 라벨.
+데이터 휘발: 24H 지난 라운드 자동 삭제. 시안에서는 좌측 끝 라운드(-22H)에 비어있는 ring 으로 "곧 사라질" 표시. 실제 구현 시 라운드가 페이지 밖으로 fade out + 우측에서 새 라운드 fade in.
+셔터 버튼 상태: NOW 라운드에서만 활성(오렌지). 과거 라운드 보고 있을 때는 비활성(페이퍼 톤, dashed) → "지금 이 시간대만 송출 가능" 의 시각 신호.
+셀 정보 4종 노출: frame number(좌상단, 필름) / 멤버 태그(좌하단, 본인은 그린) / 새 송출 LED 도트(우상단, 호흡) / 영상 길이 배지(우하단, ▶). 빈 칸 = dashed + 큰 이니셜 + NO SIGNAL.
