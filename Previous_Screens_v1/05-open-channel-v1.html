OPEN CHANNEL V1 · 3 PHASE FLOW
+3 phase: SETUP (채널 이름 + 닉네임 + 컬러 선택) → BROADCASTING (자동 발급 1.5초) → READY (초대 + ENTER). Tune In 의 짝, 디자인 시스템(페이퍼/모노/잉크) 그대로 재사용.
+SETUP 입력 3개: 채널 이름 16자, 닉네임 4자(영문 모노 라벨로 사용), 컬러 9개 chip grid (Post Detail 멤버 컬러 시스템 동일). 모두 입력되어야 OPEN CHANNEL 버튼 활성.
+컬러 chip: 9개 사각 (라운드 0px), 선택 시 paper-bg 1px gap + 잉크 2px outer ring 으로 더블 ring. preview-chip 에서 YOU 칩이 선택한 컬러로 즉시 미리보기 → 멤버 컬러가 "내 식별자"임을 학습.
+BROADCASTING phase: 4자리 슬롯 머신 회전. 80ms 간격으로 random 0-9. 좌→우 순서로 한 자리씩 잠금 (각 350ms 간격). 잠금 시 그린 ring flash 380ms (slot-lock-flash 와 동일 모션). 진행률 progress bar 동기화. 1.5초 후 자동 READY phase 진입.
·READY phase: 큰 주파수 디스플레이 (56px 모노, "."만 그린 강조). 라벨 "친구에게 이 4자리만 알려주세요" → 주파수 자체가 진입 식별자. 별도 초대 코드 없음 (Tune In 화면에 주파수 4자리 외에 다른 입력 칸이 없으므로 redundant).
+주파수 액션 2개: COPY (현재 발급된 주파수만 클립보드, 예: "107.4") / SHARE (native share sheet, 채널 이름 + 주파수 안내 메시지). 둘 다 잉크 fill 버튼, 누르면 그린으로 confirm flash. ENTER CHANNEL 만 TE Orange CTA 단독.
+주파수 발급 (시안): 100.0 ~ 119.9 범위 random. 실제 시스템에서는 서버에서 충돌 체크 + 24H 자동 휘발이 conflict 완화. 사용자 화면에서는 주파수 4자리 만으로 명확하게 표현.
·다음 검토: (a) 동일 주파수 충돌 시 host 에게 재발급 옵션, (b) 닉네임 영문 4자 강제 validation, (c) READY 에서 ENTER 누르면 Feed (1명, n=1 풀스크린)로 전환, (d) SHARE 메시지에 deep link 포함 (frequency://107.4 → 자동 Tune In).
