+구조 결정: 일반적 시간순 단일 피드(인스타/setlog)가 아닌 "사람별 칸 분할 모자이크". 방 안의 각 멤버가 자기 칸을 차지 → 채팅 없이도 "누가 있고 누가 뭘 보냈는지"가 한 화면에서 즉각.
+인원수 자동 적응 (예정): 1명 풀스크린 / 2명 1x2 / 3명 1+2 비대칭 / 4명 2x2 / 5~6명 비대칭 모자이크 / 7~9명 3x3 균등. 이번 시안은 4명 = 2x2, 4:5 필름 사진 비율.
+셀 상태 4종을 한 화면에 모두 노출: (1) 본인 YOU = 그린 라벨, (2) 다른 멤버 신규 송출 = 우상단 그린 LED 도트 호흡 펄스, (3) 영상 콘텐츠 = 좌하단 ▶ + 길이 배지, (4) 빈 칸 NO SIGNAL = dashed 테두리 + 큰 이니셜.
+필름 카메라 메타포: 좌상단 frame number (#024). 무전기 메타포: 우상단 그린 LED 도트, 상단 헤더 "4/9 LIVE + 펄스 도트". 두 메타포가 같은 셀 위에서 겹치는 게 이 앱의 정체성.
+캡션 처리: 사진 위 텍스트 오버레이 최소화 → 사진 아래 한 줄 (이름 좌, 시간 우). 페이퍼 사진첩 + 모노 캡션. 시간은 모노 약식 ("3'", "1H", "—").
·셔터 버튼: 외곽 3px 오렌지 ring + 안쪽 오렌지 dot, 사이에 페이퍼 gap. 누르면 안쪽 dot이 작아지면서 셔터가 닫히는 듯한 inset 변화. 화면 내 유일한 컬러 액센트 위치 유지.
+네비 좌(멤버 아이콘) / 가운데(셔터) / 우(채널 정보) 3개 버튼. SVG 1.7px stroke, 잉크 블랙. nav 버튼은 페이퍼 edge 1.5px outline, 누르면 raised paper로 살짝 올라옴.
