DESIGN NOTES
카테고리Nature(8) + Object(8) + Creature(6) + Symbol(7) = 29. 자연·사물·생물 + 도형/포커 문양으로 성격 스펙트럼을 커버.
라인 스타일모든 아이콘 stroke 1.5px, round cap/join 통일. 작은 사이즈(28px)에서도 실루엣이 남도록 내부 디테일 최소화.
컬러 팔레트Paper(크림) / Orange(키) / Ink(다크) / Brick(테라코타) / Navy(딥블루) / Sage(세이지). 웜 3개 + 쿨 2개 + 뉴트럴 1개 밸런스.
조합 수29 × 6 = 174. 8명 룸에서 임의 조합 시 정확히 겹칠 확률은 낮지만, 아이콘만 같을 확률은 존재함 → 배경색까지 보여서 분별 유도.
폴백선택 전 기본값은 현재의 모노그램(이름 첫 글자 + 채도 낮은 랜덤 컬러)로 유지. 선택 유도는 Settings에 "프로필 아이콘" 항목으로 노출.
접근 위치Settings > 내 프로필 → 프로필 아이콘 탭 시 본 픽커가 하단 시트로 올라옴. 저장 즉시 모든 룸의 아바타·스탬프에 반영(전역 프로필).
